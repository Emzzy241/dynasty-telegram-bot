## dynasty_telegram_bot
#### A Telegram bot made by Dynasty.

## Technologies Used
* Node.js
* Express
* Javascript
* Telegram API

## Description
_This is a Telegram bot made by Dynasty whose purpose is to respond to messages that have the key word: dynasty_

## Setup/Installation
1. Clone the Repository
```sh
    git clone https://github.com/Emzzy241/dynasty-telegram-bot.git
```

2. Install all dependencies and packages
```sh
    npm install
```

3. Run the Project
```sh
    npm run start
```
